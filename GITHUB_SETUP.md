# GitHub setup (one-time)

Local git is initialized and committed. Remote is set to:

`https://github.com/facelssfemme/recall-system.git`

## If the repo does not exist yet on GitHub

1. Open: https://github.com/new?name=recall-system&description=The+Recall+System
2. Owner: **facelssfemme**
3. Repository name: **recall-system**
4. Public
5. Do **not** add README, .gitignore, or license (repo must stay empty)
6. Click **Create repository**

## Push from this folder

```bash
cd recall-system
git push -u origin main
```
